What is gradient descent and how does it work?
Gradient descent is an optimization algorithm that minimizes a loss function by iteratively moving in the direction of steepest descent [1]. It computes gradients of the loss with respect to each model parameter, then updates parameters opposite to the gradient direction [2].

The update rule is: θ = θ − α · ∇J(θ), where α is the learning rate [1].
